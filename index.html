Bem-vindo ao site das Bandeiras dos países.

Este é um exercício que estou fazendo para treinar posicionamento de elementos utilizando somente código CSS puro e HTML para desenhar as bandeiras de alguns países.

Alguns conceitos que estou treinando com esse site incluem o uso de seletores combinados do CSS, uso dos recursos de posicionamento de elementos HTML, uso correto de elementos do HTML e de classes e Id`s, uso do z-index.

Este é um site com objetivos meramente educativos. A idéia é baseada no site original de Michael Lascarides.
